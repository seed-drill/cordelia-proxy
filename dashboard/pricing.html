Frequently Asked Questions

What is L1 and L2 memory?

L1 (hot context) is loaded at the start of every Claude session - it contains your identity, preferences, and current focus. L2 (warm index) is searchable memory that Claude can pull on demand, perfect for storing learnings, entities, and session summaries.

Is my data encrypted?

Yes. All L2 entries are encrypted at rest using AES-256-GCM with scrypt key derivation. Your encryption key never leaves your environment.

Can I self-host?

Absolutely. Cordelia is open source and designed for self-hosting. The Starter tier is completely free for local deployments. Cloud tiers include managed hosting and support.

What MCP clients are supported?

Cordelia works with any MCP-compatible client including Claude Code, Claude Desktop, and custom integrations. We provide first-class support for Claude Code with automatic hook integration.
